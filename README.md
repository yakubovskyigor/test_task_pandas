# TEST_APP_PANDAS
___

## Installation

You can install this app on your working machine from source code:
```bash
# Step -- 1.
git clone --depth=1 --branch=main 'https://github.com/yakubovskyigor/test_app_pandas.git'

# Step -- 2.
cd ./test_app_pandas/

# Step -- 3.
pip install -r requirements.txt
```
